To get in touch with me, use [Twitter](https://twitter.com/ConradIrwin) or Github Issues.

How to contribute
=================

Git clone

```
git clone git@github.com:ConradIrwin/howtobuilda.bike
```

Bundle install

```
cd howtobuilda.bike
bundle install
```

Run foreman

```
bundle exec foreman start
```
